Final Project

Description

For the final project, I used Scratch to create my game. Through Scratch, I refreshed my knowledge of previously learned topics such as forever loops, if loops, broadcasting, etc. by incorporating them into my game. The user interacts with the game by controlling Cookie Monster using the arrow keys to collect the cookies; you must collect all of the cookies on the screen before time runs out.
